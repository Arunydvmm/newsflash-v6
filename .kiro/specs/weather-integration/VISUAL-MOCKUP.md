## Color & Typography

### Location Header
- **Background:** Glass-morphism (semi-transparent with blur)
- **Text Color:** Light (for dark theme)
- **Icon:** 📍 (location pin emoji)
- **Font:** Sans-serif, medium weight

### City Name
- **Size:** Large (24px desktop, 18px mobile)
- **Weight:** Bold (700)
- **Color:** White/Light

### Country
- **Size:** Medium (16px desktop, 14px mobile)
- **Weight:** Regular (400)
- **Color:** Light gray

### Coordinates
- **Size:** Small (12px)
- **Weight:** Regular (400)
- **Font:** Monospace
- **Color:** Muted gray

### Timezone
- **Size:** Small (12px)
- **Weight:** Regular (400)
- **Color:** Muted gray

### Last Updated
- **Size:** Small (11px)
- **Weight:** Regular (400)
- **Color:** Very muted gray
- **Format:** Relative time (e.g., "2 mins ago")

---

## Interaction States

### Hover (Desktop)
- Location header background slightly brightens
- "Change Location" button becomes more prominent
- Cursor changes to pointer on button

### Active (Mobile)
- Location header has subtle highlight
- "Change Location" button shows active state
- Touch feedback visible

### Loading
- Location info shows skeleton/placeholder
- "Updated" timestamp shows "Loading..."
- Spinner visible

### Error
- Location shows "Unable to determine location"
- Fallback to "Your Location"
- Error message displayed
- "Change Location" button remains functional
